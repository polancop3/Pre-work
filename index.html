Welcome

My name is Pedro Polanco; I was born in the Dominican Republic and raised in Washington heights. The first time I came across code was when I was in high school. At the time, I wanted to learn how a calculator worked. With this initial spark of interest, i searched on google how to build a calculator, which led me into a website that had the information I needed. I dont remember if the calculator I made was written in C or C++; still, I remember for sure that the code I wrote was working and doing basic arithmetic operations, which was very cool, and I was excited with the concept of automating tasks. Still, at that time, I felt that coding was very limited since I did not know anything about it. I thought that doing a calculator was all it useful for, so I stopped coding until later on when I realized that coding is the key to a world of unlimited possibilities for the human race.
